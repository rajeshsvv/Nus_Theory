1. Install git.
2. Go the folder where your code is. 
3. Open a terminal there(windows-shift+right click, Command Prompt/Unix-right click, Terminal)
4. Type git init and press enter.
5. Open Github and signup there.
6. Login and Create a new repository.
7. Do not add readme and licence U can add it later. 
8. Copy the line git remote add origin .........................
9. Go to your local code folder open terminal or command prompt and paste that line and press enter.
10. Type git add --all and press enter.
11. Type git commit -m "Initial commit" and press enter.
12. Type git push -u origin master.
13. Give your github userid and Password.
14. Open github and open your repository and check if your code is pushed or not.
